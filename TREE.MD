## 📋 專案架構說明

### 🏗️ 核心架構
- **app/**: Angular 應用程式主目錄，包含所有業務邏輯和元件
- **assets/**: 靜態資源目錄，包含圖片、樣式、語言包等
- **environments/**: 環境配置檔案，用於開發和生產環境的差異化配置
- **styles/**: 全域樣式檔案，定義應用程式的整體視覺風格

### 🔧 主要功能模組
1. **core/**: 核心服務和功能，提供全域可用的服務
2. **layout/**: 佈局元件，定義應用程式的整體結構
3. **routes/**: 路由模組，包含所有功能頁面和業務邏輯
4. **shared/**: 共享元件和工具，可在多個模組中重複使用

### 🎯 業務功能模組
- **contract/**: 合約管理系統
- **dashboard/**: 儀表板和數據分析
- **passport/**: 用戶認證和授權
- **workspace/**: 工作區和任務管理
- **tree/**: 樹狀結構管理
- **pro/**: 專業版功能展示

### 🔥 Firebase 整合
專案已整合 Firebase 服務，包括：
- 用戶認證 (Authentication)
- 資料庫 (Firestore)
- 檔案儲存 (Storage)
- 雲端函數 (Functions)

### 📱 響應式設計
- 支援多種螢幕尺寸
- 使用 ng-zorro-antd 元件庫
- 提供深色和淺色主題
- 支援多語言國際化
